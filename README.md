### Warp In a New Project

Initialize a new nexus of development. You must choose your AI Templar.

```bash
forge init my-project --ai claude
```

*Supported Templars: `claude`, `gemini`, `cursor-agent`, `windsurf`, `qwen`, `opencode`, `codex`, `auggie`, `roo`.*

### Command the Templars

Once initialized, your Agent will recognize the following High Templar commands:

- **`/forge.plan`**: Draft the strategic implementation plan.
- **`/forge.tasks`**: Break the plan into granular, atomic tasks.
- **`/forge.implement`**: Execute the tasks using the TDD cycle.
- **`/forge.constitution`**: Establish the project's governing laws.
